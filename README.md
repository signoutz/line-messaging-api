# 🌊 ระบบแจ้งเตือนระดับน้ำ (Flood Alert System)

ระบบติดตามและแจ้งเตือนระดับน้ำอัตโนมัติผ่าน LINE โดยดึงข้อมูลจากสถานีตรวจวัดของศูนย์อุทกวิทยาและการประยุกต์ มหาวิทยาลัยเชียงใหม่ (CMU CCDC)

## ระบบทำอะไรได้บ้าง

- ตรวจสอบระดับน้ำจากสถานีตรวจวัดทั่วประเทศ แบบ realtime
- ส่งการ์ดแจ้งเตือนผ่าน LINE อัตโนมัติ เมื่อระดับน้ำเกินค่าที่กำหนด (แจ้งเตือนตาม %)
- ตั้งเวลาแจ้งเตือนรายชั่วโมงสำหรับแต่ละสถานีได้ (แจ้งเตือนตามเวลา — ไม่สน % ส่งทุกระดับ)
- ตั้งเงื่อนไขแจ้งเตือนหลายระดับต่อสถานี (Dynamic Threshold) เช่น >80% ทุก 60 นาที, >100% ทุก 5 นาที
- เลือกเปิด/ปิดการแจ้งเตือนแยกรายสถานีได้
- ส่งแจ้งเตือนไปหลายกลุ่ม LINE พร้อมกันได้
- ส่งรายงานสรุปประจำวัน (Daily Summary)
- มีหน้า Admin สำหรับจัดการทุกอย่างผ่านเว็บ

## ระดับการแจ้งเตือน

| ระดับ | ความจุลำน้ำ | ความหมาย |
|-------|------------|----------|
| 🟢 ปกติ | ≤80% | สภาวะน้ำปกติ ไม่ส่งแจ้งเตือน (ยกเว้นแจ้งเตือนตามเวลา) |
| 🟡 เฝ้าระวัง | >80% ถึง <100% | ติดตามใกล้ชิด เตรียมพร้อมอพยพ |
| 🔴 วิกฤต | ≥100% | น้ำเอ่อล้นตลิ่ง แจ้งเตือนอพยพทันที |

## Cron Jobs

| ไฟล์ | รอบ | หน้าที่ |
|------|-----|---------|
| `cron_flood.php` | ทุก 10 นาที (`*/10 * * * *`) | เช็ค % น้ำ ถ้าเกิน threshold + cooldown ผ่าน → ส่งแจ้งเตือน |
| `cron_flood_scheduled.php` | ทุกชั่วโมง (`0 * * * *`) | เช็คชั่วโมงปัจจุบัน ถ้าตรงกับเวลาที่ตั้งไว้ → ส่งข้อมูลสถานีนั้น (ทุกระดับ) |
| `cron_flood_summary.php` | 08:00 และ 16:00 (`0 8,16 * * *`) | ส่งตารางสรุปทุกสถานีไปยังกลุ่มที่ตั้งค่าไว้ |

## วิธีใช้งานหน้า Admin

เข้าได้ที่ `admin_flood.php` แล้วกรอกรหัสผ่าน

### 📡 แท็บ จุดตรวจวัด

แต่ละสถานีแสดงเป็นการ์ด ประกอบด้วย:

- **ส่วนหัว** — ชื่อสถานี, ปุ่มเปิด/ปิด 🔔/🔕, ข้อมูล live (ระดับน้ำ, ขอบตลิ่ง, %), URI link
- **เงื่อนไขแจ้งเตือน (%)** — ตั้งหลายเงื่อนไขได้ เช่น >80% → ทุก 60 นาที, >100% → ทุก 5 นาที
- **แจ้งเตือนตามเวลา** — เลือกชั่วโมงที่ต้องการส่งแจ้งเตือน (00:00 - 23:00) กดเพิ่ม/ลบได้ ส่งทุกระดับโดยไม่สน %
- **LINE Groups** — เลือกกลุ่ม LINE ที่จะรับแจ้งเตือนของสถานีนี้

### 💬 แท็บ กลุ่ม LINE

- กด **"Scan จาก Log"** เพื่อค้นหากลุ่มใหม่ที่ Bot เข้าร่วม
- กด **"ดึงชื่อกลุ่ม"** เพื่ออัปเดตชื่อกลุ่มจาก LINE
- ตั้งชื่อกลุ่มเองได้เพื่อให้จำง่าย

### 🔔 แท็บ ประวัติแจ้งเตือน

- ดูรายการแจ้งเตือนที่ส่งไปแล้วล่าสุด 50 รายการ

### 📖 แท็บ คู่มือ

- ภาพรวมระบบ, flow การทำงาน, เกณฑ์ระดับน้ำ, วิธีตั้งค่าต่างๆ

## วิธีเพิ่ม Bot เข้ากลุ่ม LINE

1. เปิดแอป LINE → ค้นหา ID **@045hnsao** แล้วเพิ่มเป็นเพื่อน
2. เข้ากลุ่มที่ต้องการ → เชิญ Bot เข้ากลุ่ม
3. ส่งข้อความอะไรก็ได้ในกลุ่ม (เพื่อให้ระบบจับ Group ID)
4. กลับมาหน้า Admin → แท็บ กลุ่ม LINE → กด "Scan จาก Log"
5. กลุ่มใหม่จะปรากฏในรายการ → นำไปผูกกับสถานีได้

## คำถามที่พบบ่อย

**ทำไมไม่ได้รับแจ้งเตือน?**
- ตรวจสอบว่าสถานีเปิดการแจ้งเตือนอยู่ (🔔)
- ตรวจสอบว่ากลุ่ม LINE ถูกผูกกับสถานีแล้ว
- ระบบมี cooldown ตามที่ตั้งค่าในเงื่อนไข — จะไม่ส่งซ้ำจนกว่าจะครบเวลาที่กำหนด

**ตั้ง Threshold เท่าไหร่ดี?**
- แนะนำ **80%** สำหรับเตือนล่วงหน้า
- หรือ **70%** ถ้าต้องการเตือนเร็วขึ้น

**แจ้งเตือนตามเวลาต่างจากแจ้งเตือนตาม % อย่างไร?**
- **ตาม %** — ส่งเมื่อระดับน้ำเกิน threshold ที่ตั้งไว้ (เช็คทุก 10 นาที)
- **ตามเวลา** — ส่งตามชั่วโมงที่กำหนด ไม่สนว่า % เท่าไหร่ ส่งทุกระดับ (เช็คทุกชั่วโมง)

**เพิ่มสถานีใหม่อย่างไร?**
- กดปุ่ม "Sync สถานี" ระบบจะดึงรายการทั้งหมดจาก API อัตโนมัติ
