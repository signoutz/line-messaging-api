# 🌊 ระบบแจ้งเตือนระดับน้ำ (Flood Alert System)

ระบบติดตามและแจ้งเตือนระดับน้ำอัตโนมัติผ่าน LINE โดยดึงข้อมูลจากสถานีตรวจวัดของศูนย์อุทกวิทยาและการประยุกต์ มหาวิทยาลัยเชียงใหม่ (CMU CCDC)

## ระบบทำอะไรได้บ้าง

- ตรวจสอบระดับน้ำจากสถานีตรวจวัดทั่วประเทศ แบบ realtime
- ส่งการ์ดแจ้งเตือนผ่าน LINE อัตโนมัติ เมื่อระดับน้ำเกินค่าที่กำหนด
- เลือกเปิด/ปิดการแจ้งเตือนแยกรายสถานีได้
- ส่งแจ้งเตือนไปหลายกลุ่ม LINE พร้อมกันได้
- มีหน้า Admin สำหรับจัดการทุกอย่างผ่านเว็บ

## ระดับการแจ้งเตือน

| ระดับ | ความจุลำน้ำ | ความหมาย |
|-------|------------|----------|
| 🟢 ปกติ | ≤80% | สภาวะน้ำปกติ ไม่ส่งแจ้งเตือน |
| 🟡 เฝ้าระวัง | >80% ถึง <100% | ติดตามใกล้ชิด เตรียมพร้อมอพยพ |
| 🔴 วิกฤต | ≥100% | น้ำเอ่อล้นตลิ่ง แจ้งเตือนอพยพทันที |

## วิธีใช้งานหน้า Admin

เข้าได้ที่ `admin_flood.php` แล้วกรอกรหัสผ่าน

### 📡 แท็บ จุดตรวจวัด

- กด **"Sync สถานี"** เพื่อดึงรายการสถานีล่าสุด
- กดปุ่ม 🔔/🔕 เพื่อเปิดหรือปิดการแจ้งเตือนแต่ละสถานี
- ตั้งค่า **Threshold** (ค่าแนะนำ 80%) — ถ้าน้ำเกินค่านี้จะส่งแจ้งเตือน
- เลือกกลุ่ม LINE ที่ต้องการให้ส่งแจ้งเตือนสำหรับแต่ละสถานี

### 💬 แท็บ กลุ่ม LINE

- กด **"Scan จาก Log"** เพื่อค้นหากลุ่มใหม่ที่ Bot เข้าร่วม
- กด **"ดึงชื่อกลุ่ม"** เพื่ออัปเดตชื่อกลุ่มจาก LINE
- ตั้งชื่อกลุ่มเองได้เพื่อให้จำง่าย

### 🔔 แท็บ ประวัติแจ้งเตือน

- ดูรายการแจ้งเตือนที่ส่งไปแล้วล่าสุด 50 รายการ

### 📖 แท็บ คู่มือ

- ภาพรวมระบบ, flow การทำงาน, เกณฑ์ระดับน้ำ, วิธีตั้งค่าต่างๆ

## วิธีเพิ่ม Bot เข้ากลุ่ม LINE

1. เปิดแอป LINE → ค้นหา ID **@045hnsao** แล้วเพิ่มเป็นเพื่อน
2. เข้ากลุ่มที่ต้องการ → เชิญ Bot เข้ากลุ่ม
3. ส่งข้อความอะไรก็ได้ในกลุ่ม (เพื่อให้ระบบจับ Group ID)
4. กลับมาหน้า Admin → แท็บ กลุ่ม LINE → กด "Scan จาก Log"
5. กลุ่มใหม่จะปรากฏในรายการ → นำไปผูกกับสถานีได้

## คำถามที่พบบ่อย

**ทำไมไม่ได้รับแจ้งเตือน?**
- ตรวจสอบว่าสถานีเปิดการแจ้งเตือนอยู่ (🔔)
- ตรวจสอบว่ากลุ่ม LINE ถูกผูกกับสถานีแล้ว
- ระบบมี cooldown 1 ชั่วโมง — จะไม่ส่งซ้ำถ้าเพิ่งแจ้งไป

**ตั้ง Threshold เท่าไหร่ดี?**
- แนะนำ **80%** สำหรับเตือนล่วงหน้า
- หรือ **70%** ถ้าต้องการเตือนเร็วขึ้น

**เพิ่มสถานีใหม่อย่างไร?**
- กดปุ่ม "Sync สถานี" ระบบจะดึงรายการทั้งหมดจาก API อัตโนมัติ
